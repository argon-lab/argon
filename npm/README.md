# argonctl

Git-like MongoDB branching for ML/AI workflows.

## Installation

```bash
npm install -g argonctl
```

## Usage

```bash
# Verify installation
argon --version

# Create a project
argon projects create --name my-project

# List projects
argon projects list

# Get help
argon --help
```
